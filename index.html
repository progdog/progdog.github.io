Pandas__How to Check if NaN Exists in Column Quicklly May 10, 2021

Data preparation

import pandas as pd

d = {
    "name": ["Kevin", "Jack", "Mary", "Bob"],
    "sex": ["Male", "Male", "Female", "Male"],
    "score": [80, 90, 95, None]
}
df = pd.DataFrame(data=d)
name sex score
Kevin Male 80.0
Jack Male 90.0
Mary Female 95.0
Bob Male NaN

Check if score column has NaN

df["score"].hasnans

Result

True
