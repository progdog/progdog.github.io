Pandas__How to Select Rows From DataFrame by conditions

import pandas as pd

d = {
    "name": ["Kevin", "Jack", "Mary"],
    "sex": ["Male", "Male", "Female"],
    "score": [80, 90, 95]
}

df = pd.DataFrame(data=d)

Single condition filter

result = df.loc[df['score']>=90]

result = df[df['score']>=90]

Pandas__How to Check if a Single Value Is NaN

2021-05-07 Pandas 

import pandas as pd

d = {'col1': [1, None], 'col2': [None, "4"]}

df = pd.DataFrame(data=d)

if pd.isna(df.iloc[0,1]):
    print("NaN")
else:
    print("Not NaN")

Result

NaN
