Pandas__How to Check if NaN Exists in Column Quicklly

2021-05-10 Pandas

Data preparation

import pandas as pd

d = {
    "name": ["Kevin", "Jack", "Mary", "Bob"],
    "sex": ["Male", "Male", "Female", "Male"],
    "score": [80, 90, 95, None]
}
df = pd.DataFrame(data=d)
name sex score
Kevin Male 80.0
Jack Male 90.0
Mary Female 95.0
Bob Male NaN

Check if score column has NaN

df["score"].hasnans

Result

True

Pandas__How to Select Column of Specific Row

2021-05-09 Pandas

Data preparation

import pandas as pd

d = {
    "name": ["Kevin", "Jack", "Mary"],
    "sex": ["Male", "Male", "Female"],
    "score": [80, 90, 95]
}
df = pd.DataFrame(data=d)
name sex score
Kevin Male 80
Jack Male 90
Mary Female 95

Select score column of filtered row

df[df['name'] == 'Mary']['score']
# OR
df[df['name'] == 'Mary'].score
# OR
df.loc[df['name'] == 'Mary', 'score']
# OR
df.loc[df['name'] == 'Mary'].score

Result

2    95
Name: score, dtype: int64
