May 09, 2021 in PANDAS

1 min read

Data preparation

import pandas as pd

d = {
    "name": ["Kevin", "Jack", "Mary"],
    "sex": ["Male", "Male", "Female"],
    "score": [80, 90, 95]
}
df = pd.DataFrame(data=d)
name sex score
Kevin Male 80
Jack Male 90
Mary Female 95

Select score column of filtered row

df[df['name'] == 'Mary']['score']
# OR
df[df['name'] == 'Mary'].score
# OR
df.loc[df['name'] == 'Mary', 'score']
# OR
df.loc[df['name'] == 'Mary'].score

Result

2    95
Name: score, dtype: int64

Select name column of 2nd row using iloc method

df.iloc[1, 0]

Result

'Jack'
