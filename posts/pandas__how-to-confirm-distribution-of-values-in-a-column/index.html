Pandas__How to Confirm Distribution of Values in a Column

2021-05-09 Pandas

Data preparation

import pandas as pd

d = {
    "name": ["Kevin", "Jack", "Mary"],
    "sex": ["Male", "Male", "Female"],
    "score": [80, 90, 95]
}
df = pd.DataFrame(data=d)
name sex score
Kevin Male 80
Jack Male 90
Mary Female 95

Confirm count of unique values (default order: descending order)

df['sex'].value_counts()

Result

Male      2
Female    1
Name: sex, dtype: int64

Sort by value count in ascending order

df['sex'].value_counts(ascending=True)

Result

Female    1
Male      2
Name: sex, dtype: int64

Confirm frequency of value

df['sex'].value_counts(normalize=True)

Result

Male      0.666667
Female    0.333333
Name: sex, dtype: float64
