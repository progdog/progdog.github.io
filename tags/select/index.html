Pandas__How to Select Column of Specific Row

2021-05-09 Pandas

Data preparation

import pandas as pd

d = {
    "name": ["Kevin", "Jack", "Mary"],
    "sex": ["Male", "Male", "Female"],
    "score": [80, 90, 95]
}
df = pd.DataFrame(data=d)
name sex score
Kevin Male 80
Jack Male 90
Mary Female 95

Select score column of filtered row

df[df['name'] == 'Mary']['score']
# OR
df[df['name'] == 'Mary'].score
# OR
df.loc[df['name'] == 'Mary', 'score']
# OR
df.loc[df['name'] == 'Mary'].score

Result

2    95
Name: score, dtype: int64

Pandas__How to Select Rows From DataFrame by conditions

2021-05-08 Pandas 
import pandas as pd

d = {
    "name": ["Kevin", "Jack", "Mary"],
    "sex": ["Male", "Male", "Female"],
    "score": [80, 90, 95]
}

df = pd.DataFrame(data=d)
name sex score
Kevin Male 80
Jack Male 90
Mary Female 95

Single condition filter

result = df.loc[df['score']>=90]
# OR
result = df[df['score']>=90]

Result

name sex score
Jack Male 90
Mary Female 95
